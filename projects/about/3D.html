-------- -CONTROLS- ---------
WASD / ARROW KEYS -- Movement
SPACE -- Reset Position
E -- Toggle UI
R -- Reset Darkness intensity to Default Values
T -- Reset Twist
- / + -- Twist
[ / ] -- Sprite
